## BlogPessoal – Frontend

Projeto frontend do BlogPessoal, desenvolvido em React, consumindo um backend já existente com o mesmo tema.

## Objetivo
Construção da interface do blog pessoal, iniciando pela estrutura principal da aplicação.

## Tecnologias

React

Typescript

Vite

Tailwind

## Estrutura Atual

Home: componente principal (main) da página inicial.

## Como executar o projeto

```
npm install
npm run dev
```
